# Dataset Information

Due to storage limitations, the full image datasets used in this project are not stored directly in this GitHub repository.

The datasets include:
- Synthetic construction site images labeled as safe and unsafe
- Real construction site images split into training and test sets

All datasets are available via Google Drive at the following link:
https://drive.google.com/drive/folders/1Hioa_m6BlWWKENiz6IDaV5BUEwPW_GtO?usp=drive_link

The provided notebooks are configured to load the data directly from Google Drive.
When running the code locally, dataset paths may require minor adjustments.
